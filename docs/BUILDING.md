# Building


## For release

Build css and js

```
npm run build-release
```

### JS

```
npm run build-release:js
```

### CSS


```
npm run build-release:css
```

## During dev

Build css and js

```
npm run build
```

### JS

```
npm run build:js
```

### CSS


```
npm run build:css
```
